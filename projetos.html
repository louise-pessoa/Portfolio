Busca Silenciosa (2025)

Projeto desenvolvido para a disciplina Projetos 1, do 1º período de Ciência da Computação no CESAR School.

Busca Silenciosa é uma adaptação da brincadeira infantil "Cabra-Cega" pensada para pessoas com deficiênca auditiva, pois a forma tradicional depende de estímulos sonoros, corroborando com a exclusão das PCD auditivas nas interações sociais e culturais.

Tecnologias utilizadas:

  • C++
  • Arduino IDE
  • ESP32
  • Markdown
